I created this project to sharpen up some Powershell skills and as a study aid for basic cryptography with OpenSSL.

It seeks to emulate a PKI and mailbox application in a sort of "diorama" fashion. It is non-functional for anything outside of demonstrative purposes.

If for some reason you want to see what this does, plop the two .ps1 files in a directory, and run Alice.ps1.
